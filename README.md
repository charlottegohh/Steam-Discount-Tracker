# Steam Discount Dashboard

A full-stack web application that scrapes and displays the latest discounted games from Steam Specials site

🔗 **Live Demo**: [steam-discount-tracker-liart.vercel.app](https://steam-discount-tracker-liart.vercel.app)

---

## Built With

**Frontend**
- React
- Vite
- Tailwind CSS
- Axios
- Lucide React

**Backend**
- Python
- Flask
- Flask-CORS
- BeautifulSoup4
- Requests
- Gunicorn

---

## Prerequisites

- Node.js (v18 or higher)
- Python (v3.10 or higher)
- pip

---

## Roadmap

- Search bar to filter games by title
- Card grid layout (4 cards per row) for a more compact and readable display
- Filter by minimum discount percentage
- Pagination or infinite scroll for more results
